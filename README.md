## Описание
REST API для системы управления задачами с JWT-аутентификацией, разработанное на Spring Boot. Позволяет создавать пользователей, аутентифицироваться и управлять задачами.
## Технологии
Java 17, 
Spring Boot 3.5.3, 
Spring Security, 
JWT Authentication, 
Spring Data JPA, 
PostgreSQL, 
Maven.
